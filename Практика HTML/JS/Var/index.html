Изменение стоимости цены товара с помощью переменной





При сложениичисел "строка и число" происходит конкатенация строк, что приводит к неправильному отображению числа. Для правильного счета нужно добавить команду Number перед 'строкой'.
Это работает только с числами, если в 'стоку' добавить буквы или слово, то получим особое значение NaN - не число.



При умножении строки и числа происходит правильное умножение (без добавления команды Number)



Используем переменную в качестве слагаемого, вычитаемого, умножения, деления
